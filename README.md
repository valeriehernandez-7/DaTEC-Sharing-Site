### About

_DaTEC is a dataset sharing platform implementing a polyglot persistence architecture with four specialized databases working cohesively. The system supports 21 user stories covering authentication, dataset management, social features, commenting, voting, and administrative controls._

### Development environment

- OS : [Windows 11](https://www.microsoft.com/en-us/software-download/windows11)
- Code Editor : [VSCode](https://code.visualstudio.com/download)
- API Testing : [Postman](https://www.postman.com/downloads/)
- Containerization : [Docker](https://www.docker.com/products/docker-desktop/)
- Frontend Framework : [Vue.js](https://vuejs.org/guide/quick-start)
- Backend Framework : [Node.js](https://nodejs.org/en/download/current)
- Databases: 
  - [MongoDB](https://www.mongodb.com/try/download/community)
  - [Redis](https://redis.io/downloads/)
  - [Neo4j](https://neo4j.com/download/)
  - [CouchDB](https://couchdb.apache.org/#download)
